# Google Apps Script — Vita Delta

Scripts auxiliares para preparar y auditar los Google Sheets DEV/TEST de Vita Delta.

## Scripts

- `validaciones_vita_delta_v3.gs`: aplica validaciones de datos de Fase 4.
- `protecciones_vita_delta_v1.gs`: aplica protecciones de Fase 5.
- `auditoria_fase6_v2.gs`: auditoría final de solo lectura antes de n8n.

## Seguridad

Estos scripts no contienen credenciales, tokens, URLs reales de Sheets, teléfonos, datos bancarios ni datos de huéspedes.
